# Portfolio Website

A modern, responsive, single-page professional portfolio website created for Abhishak Chaturvedi.

## Description
This project showcases personal information, education, a downloadable resume, work experience, projects, training, professional certificates (featured in an automatic sliding carousel), and contact details. It features a clean design inspired by microsoft.ai with smooth scrolling, fade-in animations, and interactive elements.

## Technologies Used
- **HTML5**: Semantic markup for the structure of the application.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development and responsive design.
- **Vanilla JavaScript**: Logic for the certificate carousel, scroll animations, mobile menu, and modal interactions without relying on heavy frameworks.
- **Lucide Icons**: Clean, consistent SVG icons used throughout the interface.
